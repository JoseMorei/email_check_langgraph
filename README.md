# Email Check LangGraph

A simple tool to check the language of email content using langgraph.

## Installation

To install the package, use the following command:

```bash
pip install email-check-langgraph
```

## Usage

Here is a simple example of how to use the tool:

```python
from email_check_langgraph import check_language

result = check_language("Hello world")
print(result)
```
